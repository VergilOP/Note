Support Vector Machines: Sequential Minimal Optimisation(SMO)

  • Solving the Optimisation Problem

    • Sequential minimal optimisation is one of the most popular techniques.
      • It breaks the quadratic programming problem into subquadratic programming problems that can be solved analytically one at a time. -> a subset of aaa.
      • Uses heuristics to decide which of these smaller problems to solve at each step -> which subset of aaa to solve
