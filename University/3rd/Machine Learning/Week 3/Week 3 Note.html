Support Vector Machines(SVMs)

  • The perpendicular distance γ\gammaγ between the decision boundary and the closest training examole is called the margin

  • The decision boundary can be chosen so as to maximise the margin

  • Training examples that are exactly on the margin are called support vectors

  • Given a set of training examples

    J={(x⃗(1),y(1)),(x⃗(2),y(2)),...,(x⃗(N),y(N))}J = \{(\vec{x}^{(1)}, y^{(1)}),(\vec{x}^{(2)}, y^{(2)}),...,(\vec{x}^{(N)}, y^{(N)})\} J={(x(1),y(1)),(x(2),y(2)),...,(x(N),y(N))}

    where (x⃗(i),y(i))∈X×Y(\vec{x}^{(i)},y^{(i)})\in X \times Y(x(i),y(i))∈X×Y are drawn from a fixed albeit unkown joint probability distribution P(x⃗,y)=P(y∣x⃗)P(x⃗)P(\vec{x},y) = P(y|\vec{x})P(\vec{x})P(x,y)=P(y∣x)P(x)

  • Goal: to learn a function ggg able to generalise to unseen(test) examples of the same probability distribution P(x⃗,y)P(\vec{x},y)P(x,y)

    • ggg: X→YX \rarr YX→Y, mapping input space to output space
    • ggg as a probability distribution approximating P(y∣x⃗)P(y|\vec{x})P(y∣x)

Convexity

  • Convex Sets

    • A set CCC is convex if the line segment between any two points in CCC lies in CCC
    • For any two points x⃗(1),x⃗(2)∈C\vec{x}^{(1)}, \vec{x}^{(2)} \in Cx(1),x(2)∈C and any λ∈(0,1)\lambda \in (0,1)λ∈(0,1), we have:

      λx⃗(1)+(1−λ)x⃗(2)∈C \lambda \vec{x}^{(1)} + (1 - \lambda)\vec{x}^{(2)} \in C λx(1)+(1−λ)x(2)∈C

  • Convex Functions

    • A convex function f(x⃗)f(\vec{x})f(x) is a function with a convex domain CCC that satisfies the following condition for any x⃗(1),x⃗(2)∈C\vec{x}^{(1)}, \vec{x}^{(2)} \in Cx(1),x(2)∈C and λ∈(0,1)\lambda \in (0,1)λ∈(0,1)

      f(λx⃗(1)+(1−λ)x⃗(2))≤λf(x⃗(1))+(1−λ)f(x⃗(2)) f\left( \lambda \vec{x}^{(1)} + (1 - \lambda) \vec{x}^{(2)} \right) \leq \lambda f\left( \vec{x}^{(1)} \right) + (1 - \lambda) f\left( \vec{x}^{(2)} \right) f(λx(1)+(1−λ)x(2))≤λf(x(1))+(1−λ)f(x(2))

    • Strictly convex: satisfies the condition with <
  • Importance of Convexity in Machine Learning/Optimisation
    • Any minimum in a convex function is a global minimum
    • A strictly convex function has at most one stationary (critical) point. If such a point exists, it is a global minimum
